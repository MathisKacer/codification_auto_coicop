Le top-3 de TTC

Justesse des rangs 2 et 3, valeur ajoutée du top-3, et cas où TTC a raison seul

Published

July 23, 2026

[charger_donnees] projet-budget-famille/data/workflow_runs/2026-06-29/codif-vvkv9/decide-coicop/predictions.parquet → shape = (5825, 38)

TTC (TorchTextClassifier) ne fournit pas un seul code mais un classement de ses trois meilleures propositions (ttc_code_1, ttc_code_2, ttc_code_3), chacune avec une confiance. Les autres analyses du site n’utilisent que le rang 1 ; cette page regarde ce qu’apportent les rangs 2 et 3, et isole les cas où TTC est le seul à trouver le bon code. Toutes les comparaisons sont faites au niveau 4 (sous-classe).

Justesse de chaque rang TTC

Part des produits pour lesquels le code d’un rang donné correspond au vrai code. La dernière ligne — le vrai code présent quelque part dans le top-3 — est le plafond que l’on atteindrait en sachant toujours choisir le bon rang.

Cas Effectif % des produits
0 Rang 1 correct 4326 74.3%
1 Rang 2 correct 475 8.2%
2 Rang 3 correct 161 2.8%
3 Vrai code présent dans le top-3 4962 85.2%

Le rang 1 seul est correct dans 74.3% des cas, mais le vrai code figure dans le top-3 dans 85.2% des cas — soit 10.9 points de couverture supplémentaire dormant dans les rangs 2 et 3. (TTC fournit toujours un rang 2 et un rang 3 : présents dans 100.0% et 100.0% des cas.)

Valeur ajoutée des rangs 2 et 3

La question opérationnelle : quand le rang 1 se trompe, à quelle fréquence le vrai code est-il quand même récupérable en rang 2 ou 3 ?

Cas (parmi les produits où le rang 1 a tort) Effectif % des cas où rang 1 a tort % de tous les produits
0 Rang 2 rattrape 475 31.7% 8.2%
1 Rang 3 rattrape 161 10.7% 2.8%
2 Rang 2 ou 3 rattrape 636 42.4% 10.9%

Sur les 1499 produits où le rang 1 de TTC se trompe, le vrai code est malgré tout présent en rang 2 ou 3 dans 636 cas (42.4%). C’est le gain potentiel qu’on récupérerait en exploitant le top-3 plutôt que le seul rang 1.

La confiance discrimine-t-elle les rangs 2/3 corrects ? Si oui, ttc_conf_2 et ttc_conf_3 sont des variables utiles pour départager. Confiance médiane selon que le rang est correct ou non :

Rang Conf. médiane si correct Conf. médiane si incorrect
0 Rang 2 0.16 0.0
1 Rang 3 0.04 0.0

TTC a-t-il raison seul ?

Cas où TTC trouve le bon code alors qu’aucun des trois autres classifieurs de base (LCS, RAG, RAG-ANN) ne l’a — la contribution propre de TTC, qu’on perdrait en le retirant.

Cas Effectif % des produits
0 Rang 1 correct, aucun autre classifieur correct 363 6.2%
1 Vrai code dans le top-3, aucun autre classifie... 431 7.4%

TTC est seul à trouver le bon code (rang 1) dans 363 cas (6.2%). En comptant son top-3, il apporte le vrai code qu’aucun autre classifieur n’a proposé dans 431 cas (7.4%).

Pour la modélisation

Ces chiffres motivent directement l’ajout de features dérivées du top-3 de TTC (cf. modélisation et le notebook SIRUS) : le rang auquel un code candidat apparaît chez TTC (ttc_rang) et la confiance associée portent un signal réel, puisque le vrai code est présent dans le top-3 bien plus souvent qu’en rang 1 seul, et que la confiance des rangs 2/3 sépare les bons des mauvais.
